# Weather Application



1 ) open terminal 

2) make directory using command below:

```
mkdir weather_app
```

3) take checkout :

```
git clone https://github.com/pl2ern4/weather_app.git .
```

4) Install all the dpendencies and start project

```
npm install && npm start
```

6) Can check using 

```
npm test 
```
